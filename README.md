# 📚 Library Management System

A comprehensive **Library Management System** built using **Java Swing** and **MySQL**. This desktop application provides role-based access for **Librarians**, **Students**, and **Faculty** members to efficiently manage library operations including book issuance, fine calculation, seat reservation, and report generation.

This project was developed as a complete library automation solution, demonstrating database integration, user authentication, and multi-role dashboard implementation using Java.

---

## 🚀 Features

### 👨‍💼 Librarian Features
- 📖 **Book Management** – Add, update, delete, and search books
- 👥 **Student & Faculty Registration** – Register new users with validation
- 📋 **Book Issue & Return** – Issue books to students/faculty with due dates
- 💰 **Fine Management** – Automatic fine calculation (₹5/day for overdue books)
- 📊 **Reports Generation** – Generate reports for books, users, fines, and overdue items
- 🪑 **Seat Monitoring** – View and monitor seat reservations in real-time
- ⏰ **Overdue Tracking** – Track and manage overdue books

### 🧑‍🎓 Student Features
- 🔍 **Search Books** – Search available books by title, department, or subject
- 🪑 **Seat Reservation** – Reserve library seats for 2-hour sessions
- 👤 **Profile View** – View personal details, fines, and issued books
- 🔐 **Change Password** – Update account password securely

### 👨‍🏫 Faculty Features
- 🔍 **Search Books** – Same book search functionality as students
- 👤 **Profile View** – View faculty details and issued books
- 🔐 **Change Password** – Update account password

### 🔐 Authentication System
- Role-based login (Librarian, Student, Faculty)
- Secure password validation
- Session management across modules

## 🛠️ Technologies Used

- **Java** – Core programming language
- **Java Swing** – GUI framework for desktop application
- **MySQL** – Database management system
- **JDBC** – Java Database Connectivity
